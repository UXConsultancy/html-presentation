# HTML Presentation

A Claude Code plugin for scaffolding full viewport, scroll snapping HTML presentation websites.

## Install

Add the UXC marketplace to your Claude Code settings, then enable this plugin:

```
claude plugin add UXConsultancy/claude-plugins
```

## Usage

Run `/create-presentation` or ask Claude to create a new presentation.

## What You Get

- Full viewport sections with scroll snap on desktop
- Fixed progress rail navigation
- Scroll reveal animations
- Counter animations for stat numbers
- SVG path drawing on scroll
- Logo colour switching for light/dark sections
- Optional password gate
- Responsive (stacks on mobile, no snap)
- Reduced motion support throughout

## Section Types

1. **Hero** — Full viewport intro with decorative SVG
2. **Stat Cards** — 3 animated stat cards on light background
3. **Feature Grid** — 2 column card grid on dark background
4. **Deep Dive** — Immersive full viewport narrative slide
5. **Media + Content** — What/How/Why with media placeholder
6. **CTA** — Call to action with outcome cards
