# /create-presentation

Scaffold a new HTML presentation website with full viewport sections, scroll snap, progress rail, and all interactions pre-wired.

## What This Creates

A complete, working presentation site with:

- 10 sections (hero, stats, features, 3 deep dives, 3 media+content, CTA)
- Full CSS design system with customisable tokens
- Scroll snap on desktop, responsive on mobile
- Progress rail navigation
- Scroll reveal animations
- Counter animations
- Optional password gate

All content is lorem ipsum, ready to customise.

## Instructions

Follow **Workflow 1: Create Presentation** from the html-presentation skill.
